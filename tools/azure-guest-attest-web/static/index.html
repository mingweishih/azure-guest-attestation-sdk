Tools

System Diagnostics

Check TPM availability, OS platform, and CVM type detection.

CVM Attestation Report

Fetch the full CVM attestation report from the TPM NV index, including report header, TEE report (SNP/TDX/VBS), and runtime claims.

Formats: hex:..., utf8:..., or raw hex string

TEE Report

Fetch and parse the raw TEE-specific report (AMD SEV-SNP, Intel TDX, or VBS).

TD Quote

Fetch and parse an Intel TDX quote from the platform (IMDS). Only available on TDX CVMs.

Isolation Evidence

Fetch platform isolation evidence: VCEK certificate chain (SNP) or TD Quote (TDX).

AK Certificate

Fetch the Attestation Key certificate from the TPM NV index. Displayed in PEM format.

AK Public Key

Fetch the Attestation Key public area from the TPM.

PCR Values

Read Platform Configuration Register values from the TPM across all available hash banks (SHA-1, SHA-256, SHA-384).

Comma-separated list of PCR indices to read

TPM Event Log

Parse the platform TPM event log and replay PCR values. Reads from standard OS locations.

Guest Attestation

Perform full guest attestation: collect CVM + device evidence, build attestation report, and submit to a provider. Returns a JWT token (MAA) or echo (loopback).

Base URL — the SDK appends the correct API path automatically
JSON object of key-value pairs

TEE-Only Attestation

Perform TEE-only platform attestation (no TPM/PCR evidence). Sends SNP report + VCEK chain or TDX quote directly to an MAA platform endpoint.

TDX Endorsement (THIM)

Query TDX endorsement data from Azure Trusted Hardware Identity Management. Supports listing known MRTDs, fetching endorsement by MRTD, or auto-detecting from the platform TDX report.

THIM region (e.g. westus, eastus, westeurope)
SHA-384 measurement — select from the list above or paste manually

Parse JWT Token

Decode a JWT attestation token (base64url dot-separated). No signature verification — only decodes header and payload for inspection.
